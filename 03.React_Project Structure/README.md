### ğŸ“„ `package.json`

Defines the **dependencies**, **devDependencies**, and **scripts** used in the project.

#### Key Sections:

* **dependencies** â€“ Libraries required at runtime (e.g., `react`, `react-dom`)
* **devDependencies** â€“ Tools used only during development (e.g., Vite, ESLint)
* **scripts** â€“ Shortcuts for common commands

#### Example:


## â–¶ï¸ Development Workflow

### Start Development Server

```bash
npm run dev
```

What happens:

1. Vite serves `index.html`
2. Browser loads `main.jsx`
3. React renders `App` into `#root`
4. Hot reload updates the browser on file changes
